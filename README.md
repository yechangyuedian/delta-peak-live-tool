# 三角洲巅峰赛工具集

三角洲行动主播巅峰赛相关工具合集，包含 OBS 排名挂件和 B 站弹幕自动发送脚本。

## 项目清单

| 工具 | 说明 |
|------|------|
| OBS 排名挂件 | Python 脚本，实时查询巅峰赛排名，配合 OBS 直播显示 |
| B站弹幕发送 | 油猴脚本，自动读取 TXT 文件或巅峰赛数据，自动发送弹幕到直播间 |

---

## 一、OBS 排名挂件

三角洲行动主播巅峰赛实时排名查询工具，配合 OBS 直播使用，自动定时更新主播排名数据。

### 功能特点

- 🎯 **精准查询**：通过官网搜索框定向搜索主播，无需爬取全量榜单
- 🔄 **自动刷新**：可配置刷新间隔，定时自动更新排名数据
- 📦 **缓存机制**：5分钟本地缓存，减少重复请求
- 🌐 **浏览器自动探测**：自动识别 Chrome / Edge / Playwright 内置浏览器
- ⚙️ **配置持久化**：主播名称和刷新间隔自动保存到 config.ini
- 📝 **原子写入**：防止 OBS 读取到不完整的文本文件

## 数据展示

OBS 读取 `obs_rank_display.txt` 文件，显示内容包括：
- 主播名称 + 平台
- 总排名
- 仓库总价值
- 击败干员数
- 破译曼德尔砖数
- 参赛总局数
- 更新时间

## 环境要求

- Python 3.8+
- Chrome 或 Edge 浏览器（推荐）

## 安装使用

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 安装 Playwright 浏览器

```bash
playwright install chromium
```

### 3. 运行程序

```bash
python obs_rank.py
```

首次运行会提示输入主播名称和刷新间隔，配置会自动保存。

### 4. OBS 配置

1. 在 OBS 中添加「文本（GDI+）」来源
2. 勾选「从文件读取」
3. 选择程序目录下的 `obs_rank_display.txt`
4. 设置合适的字体、颜色和位置

### 文件说明

| 文件 | 说明 |
|------|------|
| `obs_rank.py` | 主程序入口，OBS 排名显示逻辑 |
| `scraper.py` | 数据抓取模块，Playwright 页面渲染与搜索 |
| `requirements.txt` | Python 依赖清单 |
| `config.ini` | 运行后生成，保存主播配置 |
| `obs_rank_display.txt` | 运行后生成，OBS 读取的文本文件 |
| `delta_rank_cache.json` | 运行后生成，数据缓存 |
| `obs_rank.log` | 运行后生成，运行日志 |

## 二、B站弹幕自动发送脚本

油猴（Tampermonkey）脚本，自动读取 TXT 文件内容或巅峰赛数据，定时发送弹幕到 B 站直播间。

### 功能特点

- 📄 **TXT 文件模式**：读取本地 TXT 文件，按行分组自动发送弹幕
- 🔄 **文件自动刷新**：每次循环重新从磁盘读取最新文件内容，无需手动重新选择
- 🏆 **巅峰赛模式**：自动查询三角洲巅峰赛选手排名，按模板发送弹幕
- ✂️ **自动分段**：超长弹幕自动拆分，单条不超过 20 字
- ⚙️ **可配置间隔**：自定义发送间隔和循环冷却时间
- 🎨 **图形界面**：自带控制面板，操作简单直观

### 安装使用

1. 先安装浏览器扩展 **Tampermonkey**（油猴）
2. 点击脚本文件 `bilibili-danmaku-sender.user.js` 安装
3. 打开 B 站直播间页面
4. 右下角会出现「📝 发弹幕」按钮，点击打开控制面板
5. 选择模式（TXT 文件 / 巅峰赛查询），配置参数
6. 点击「开始发送」即可

### TXT 文件模式

- 支持 2 行合并为一组发送
- 每发送 3 组后进入冷却，自动重新读取文件最新内容
- 推荐使用 Chrome/Edge 浏览器的「自动刷新模式」，文件修改后自动同步

### 巅峰赛模式

- 需要先打开并登录 [三角洲巅峰赛官网](https://df.qq.com/cp/a20260611dfs/)
- 支持自定义弹幕模板，可用变量：`{rank}`（排名）、`{asset}`（资产）
- 可开启自动刷新，定时查询最新数据

### 文件说明

| 文件 | 说明 |
|------|------|
| `bilibili-danmaku-sender.user.js` | B站弹幕发送油猴脚本 |

```bash
pyinstaller -F -n delta-peak-rank obs_rank.py
```

## 注意事项

- 建议刷新间隔不低于 5 分钟，避免给官网服务器造成压力
- 本工具仅供学习交流使用，请遵守目标网站的使用条款
- 主播名称需与巅峰赛榜单上显示的名称完全一致
